# Simple Date

一个可爱、可交互的约会邀请网页，使用原生 HTML、CSS 和 JavaScript 构建，无需安装依赖。

## 本地预览

直接打开 `index.html`，或在当前目录运行：

```powershell
python -m http.server 8000
```

然后访问 <http://localhost:8000>。

## 个性化链接

可以通过 URL 参数预设昵称：

```text
https://你的用户名.github.io/仓库名/?from=小简&to=阿满
```

## 发布到 GitHub Pages

1. 将项目推送到 GitHub 仓库。
2. 打开仓库的 **Settings → Pages**。
3. 在 **Build and deployment** 中选择 **Deploy from a branch**。
4. 选择 `main` 分支和 `/ (root)` 目录后保存。

几分钟后即可通过 GitHub Pages 地址访问。
